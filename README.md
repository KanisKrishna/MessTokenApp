# MessTokenApp

An Android application to manage hostel mess food tokens digitally.

## 🔹 Features
- Splash screen (MESS)
- Home screen with Token & Admin tabs
- Meal selection (Breakfast, Lunch, Snacks, Dinner)
- Prevents duplicate token for the same meal
- Admin can manage student entries
- Firebase integration (optional)

## 🔹 Tech Stack
- Android (Java)
- Firebase Firestore (optional)

## 🔹 How to Run
1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or Android phone
